The guarantee

The achievable promise is “no ungrounded claim,” not “zero hallucination.”

Every answer is checked, per claim, against the evidence it cites. A claim the evidence doesn't support is removed; when nothing supports the question, the system says so. Confidence is expressed as structured signals — supporting sources, conflicts, languages in evidence — never an uncalibrated score.

01 · Ground

Cite or abstain

Answers are generated at temperature 0 from retrieved passages only, quoting verbatim in the source language. Weak, missing, or conflicting evidence → an honest refusal.

02 · Verify

Per-claim faithfulness

A grounding gate rejects any answer word not present in the cited passage. Unsupported claims are dropped before the answer is returned; conflicts are surfaced, not hidden.

03 · Trace

Provenance to the pixel

Each Evidence Unit carries a bbox-level citation — document, page, region — so a claim resolves back to the exact spot on the page it came from.

The library

Three verbs. Models are yours.

CiteNexus bundles no models — embedding, LLM, reranker, and vision are injected OpenAI-compatible endpoints. The library owns orchestration, storage, retrieval, fusion, grounding, and evaluation.

quickstart.py
# ingest → ask → evaluate, over your own S3 bucket
from citenexus import CiteNexus

rag = CiteNexus("s3://my-bucket", signals=["embedding", "text"])

rag.ingest()                        # any artifact type; sync or async
answer = rag.ask("Can the employee disclose this?")
score  = rag.evaluate("golden.csv")   # scored + audited
ingest()

Any artifact → cited evidence

PDF, DOCX, PPTX, XLSX, HTML, Markdown, CSV, images-into-evidence. Idempotent by content hash, signal-gated, S3-native.

ask()

Grounded, in the query's language

Strict by default. The answer is guaranteed to be in the language you asked in, cited to the evidence it rests on.

evaluate()

Prove it, per language

Groundedness, citation, and refusal metrics against a golden set — append-only audit, offline judge baseline.

How it holds the line

Navigate freely — cite precisely.

Conditional vision

A pre-filter routes each image to text, OCR, vision, or skip — so decorative art never burns a model call, and real figures become citable evidence.

Graph & wiki

Navigation over evidence, not a second source of truth: every graph hop and wiki hit resolves down to a bbox-cited Evidence Unit before an answer is formed.

Multilingual

Passages stay verbatim in their own language; any translation is additive and marked. The answer comes back in the language of the question.

Access-scoped

Scope resolves to a hard partition pre-filter before retrieval — evidence outside the caller's scope is never even a candidate.

Honest boundaries

A small core, on purpose.

The discipline that makes the guarantee credible is knowing what to leave out. Everything here is a deliberate line, not a missing feature.

It is

  • An evidence-first RAG library — a reference you build products on.
  • An ingest engine for real artifacts, resolving every hit to a cited Evidence Unit.
  • Model-agnostic: bring your own embedding, LLM, reranker, and vision endpoints.
  • S3-native storage with reproducible, auditable provenance.

It is not

  • Not a model host — nothing is bundled; every model is injected.
  • Not a code-comprehension or call-graph tool — guessed edges would betray cite-or-abstain.
  • Not a memory or “brain,” and not an end-user app — those are separate repos built on top.
  • Not a promise of zero hallucination — a promise of no ungrounded claim.
