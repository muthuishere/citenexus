# github.com/muthuishere/citenexus/golang

[![Go Reference](https://pkg.go.dev/badge/github.com/muthuishere/citenexus/golang.svg)](https://pkg.go.dev/github.com/muthuishere/citenexus/golang)

Go port of CiteNexus, a multilingual RAG library that answers only when the retrieved evidence is strong.

This directory has no root package of its own — see the subpackages (`ingest`, `core`, `graph`, `models`, `storage`, ...) for the actual API surface, and the [repo README](../README.md) for the full project layout and build plan.
