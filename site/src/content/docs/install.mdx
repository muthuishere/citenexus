## What each port gives you today

- **Python** — the reference facade. Inject any OpenAI-compatible embedding + LLM
  (+ optional reranker and vision), point it at a local folder or an S3 bucket,
  and call `ingest` / `ask` / `evaluate`.
- **Go & JavaScript** — the deterministic cite-or-abstain **core** at byte-for-byte
  parity: extraction (PDF/DOCX/PPTX/XLSX/HTML/MD/CSV/txt), language detection,
  chunking, BM25 + RRF fusion, the grounding gate, and `ask` over a corpus.

Every deterministic decision is pinned by the shared
[`conformance/`](https://github.com/muthuishere/citenexus/tree/main/conformance)
suite. Head to the [Quickstart](/citenexus/quickstart/) next.
