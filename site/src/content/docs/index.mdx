## Answer only from evidence. Otherwise abstain.

In law, medicine, and compliance, a confidently wrong answer isn't the model's
mistake — it's yours, and you're the one who has to defend it. CiteNexus answers
**only** from retrieved evidence, quotes it verbatim with a page and bbox, and
**abstains** when the evidence is weak, missing, or conflicting. The same
guarantee, in every language:

<Tabs syncKey="lang">
<TabItem label="Go">
```go
package main

import (
	"fmt"

	"github.com/muthuishere/citenexus/golang/answer"
)

func main() {
	corpus := []answer.Doc{
		{DocumentID: "employee-nda", Text: "The employee shall not disclose confidential information."},
	}

	res := answer.Ask(corpus, "Can the employee disclose confidential information?", answer.DefaultTopK)
	fmt.Println(res.Answer)            // the verbatim quote — or the pinned refusal
	fmt.Println(res.Evidence.Decision) // "answered" | "refused"
}
```
</TabItem>
<TabItem label="JavaScript">
```ts
import { ask } from "@muthuishere/citenexus"

const corpus = [
  { document_id: "employee-nda", text: "The employee shall not disclose confidential information." },
]

const r = ask(corpus, "Can the employee disclose confidential information?")
console.log(r.answer)            // the verbatim quote — or the pinned refusal
console.log(r.evidence.decision) // "answered" | "refused"
```
</TabItem>
<TabItem label="Python">
```python
from citenexus import CiteNexus

rag = CiteNexus("./citenexus-data", embedder=..., generator=...)  # your OpenAI-compatible models

rag.ingest(text="The employee shall not disclose confidential information.", document_id="employee-nda")

r = rag.ask("Can the employee disclose confidential information?")
print(r.answer)             # the verbatim quote — or the pinned refusal
print(r.evidence.decision)  # "answered" | "refused"
```
</TabItem>
</Tabs>

<Aside type="note" title="Same decision, different surface">
All three answer the same question from the same passage and print the same
decision. The difference is the surface: **Go and JavaScript** expose the
hermetic cite-or-abstain **core** — `ask` over an in-memory corpus, plus
extraction, language detection, chunking, BM25 + RRF fusion, and the grounding
gate. **Python** is the batteries-included library: your injected models,
S3-native storage, and the `ingest` / `ask` / `evaluate` verbs (there is no
`CiteNexus` facade in Go/JS yet). The deterministic decision is byte-for-byte
identical, pinned by the shared
[`conformance/`](https://github.com/muthuishere/citenexus/tree/main/conformance)
suite — see the full Python setup in the [Quickstart](/citenexus/quickstart/).
</Aside>

## More than retrieval

Dense + lexical + structure retrieval is the base. On top, CiteNexus builds an
entity [graph](/citenexus/graph/) and a distilled, cross-linked
[wiki](/citenexus/wiki/) — both **navigate-not-cite**: you browse the map, but
every hit resolves down to a real cited passage before an answer is generated.

## Proven on a real corpus

The [law worked example](/citenexus/benchmark-law/) runs this live on California
landlord–tenant statutes (Jina + Gemini, nothing mocked): **100% groundedness and
citation, zero fabricated answers** — with an honest look at where relevance-only
ranking still falls short.

## Polyglot, at parity

Not a wrapper: every deterministic decision is byte-for-byte identical across
Go, JavaScript, and Python, pinned by a shared conformance suite. Install for
yours on the [install page](/citenexus/install/), then walk the
[quickstart](/citenexus/quickstart/).
