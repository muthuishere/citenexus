Quickstart & recipes

Point CiteNexus at your documents — on disk or on S3 — bring your own models, and get answers that cite their source or abstain. The Python API is shown here; Go and JS reproduce the same deterministic core.

Install

Gogo get github.com/muthuishere/citenexus/golang
JS / TSnpm i @muthuishere/citenexus
Pythonpip install citenexus

CiteNexus bundles no models. You inject any OpenAI-compatible embedding + LLM endpoint (Ollama, OpenAI, Gemini's compat endpoint, a local vLLM) — see bring your own models.

Quickstart

Ingest a file, ask a question, read the cited answer — over a local directory.

quickstart.py
from citenexus import CiteNexus
from citenexus.embed.client import OpenAICompatibleEmbedding
from citenexus.answer.generator import OpenAICompatibleGenerator

rag = CiteNexus(
    "./citenexus-data",                     # a local directory
    embedder=OpenAICompatibleEmbedding(
        base_url="http://localhost:11434/v1", model="bge-m3"),
    generator=OpenAICompatibleGenerator(
        base_url="http://localhost:11434/v1", model="qwen2.5"),
)

rag.ingest("employee-nda.pdf")              # a file on disk

answer = rag.ask("Can the employee disclose confidential information?")

print(answer.answer)                          # the verbatim quote — or a refusal
print(answer.sources[0].document, answer.sources[0].page)
If the evidence doesn't support the question, answer.evidence.decision is refused and no source is fabricated. That refusal is the guarantee — see Ask & abstain.

File-based storage

Pass a local path as the store. Ingest individual files, a folder of them, or raw text.

file_based.py
rag = CiteNexus("./citenexus-data", embedder=..., generator=...)

rag.ingest("contracts/nda.pdf")             # PDF, DOCX, PPTX, XLSX, HTML, MD, CSV, images
rag.ingest(text="Clause 4: 90 days notice.", document_id="clause-4")

S3-native storage

Point CiteNexus at a bucket with the S3 location. Credentials come from the standard AWS environment variables; endpoint_url targets MinIO or any S3-compatible store.

s3_native.py
from citenexus import CiteNexus, S3

rag = CiteNexus(
    S3(bucket="my-docs", endpoint_url="http://localhost:19000"),
    embedder=..., generator=...,
)

rag.ingest()                                  # ingest everything already in the bucket
answer = rag.ask("What notice does termination require?")
AWS_ACCESS_KEY_ID / AWS_SECRET_ACCESS_KEY (and region) are read from the environment — CiteNexus never takes a credential as an argument.

Bring your own models

Every model is an injected, OpenAI-compatible endpoint — embedding, LLM, and optionally a reranker and a vision model for figures. Nothing is bundled; the key stays in your endpoint config.

models.py
from citenexus import CiteNexus
from citenexus.embed.client import OpenAICompatibleEmbedding
from citenexus.answer.generator import OpenAICompatibleGenerator
from citenexus.vision.client import OpenAICompatibleVision

rag = CiteNexus(
    "./citenexus-data",
    embedder=OpenAICompatibleEmbedding(base_url="…/v1", model="bge-m3"),
    generator=OpenAICompatibleGenerator(base_url="…/v1", model="qwen2.5"),
    vision=OpenAICompatibleVision(base_url="…/v1", model="qwen2.5-vl"),   # figures → cited evidence
)

Ingest anything

A file, raw text, a URL, or a whole bucket — all idempotent by content hash, so re-ingesting unchanged content is a no-op.

ingest.py
rag.ingest("report.pdf")                     # a file
rag.ingest(text="Clause 4: …", document_id="c4")  # raw text
rag.ingest("https://example.com/policy")     # a URL → fetched as HTML
rag.ingest()                                  # everything in the bucket / directory

Ask & abstain

Strict by default: the answer is generated only from retrieved passages, quoted verbatim in the source language, and every claim is checked against its citation. When nothing supports the question, CiteNexus refuses.

ask.py
answer = rag.ask("Can the employee disclose this?")   # mode=strict by default

if answer.evidence.decision.value == "answered":
    src = answer.sources[0]
    print(answer.answer, "—", src.document, "p.", src.page, src.bbox)
else:
    print("abstained:", answer.answer)          # refused, not fabricated

Evaluate

Score a corpus against a golden set — groundedness, citation, and refusal metrics, per language, with an append-only audit trail.

evaluate.py
report = rag.evaluate("golden.csv")         # question,expected_document,… rows
print(report)                                 # grounded / cited / refused rates
More depth and the runnable multilingual example live in the repository — the example/ folder ingests a real corpus over MinIO + Ollama end to end.
